CSV++ - User Instructions

File Menu

  • New (Ctrl+N) - Create a new empty CSV file
  • Open (Ctrl+O) - Open an existing CSV file
  • Save (Ctrl+S) - Save the current file
  • Close - Close the current file
  • Exit - Exit the application

Settings Menu

  • Encoding - Choose file encoding (ANSI, UTF-8, UTF-16)
  • Separator - Choose column separator (Comma, Semicolon, Tab, Custom)
  • Language - Switch between English and Serbian

Toolbar Buttons

  • New - Create a new empty table
  • Open - Open an existing CSV table
  • Save - Save current table to CSV file
  • Undo/Redo - Undo or redo your last actions
  • Add Row Below - Insert a new row below the selected cell
  • Add Row Above - Insert a new row above the selected cell
  • Delete Row - Delete the selected row
  • Add Column Left - Insert a new column to the left of selected cell
  • Add Column Right - Insert a new column to the right of selected cell
  • Delete Column - Delete the selected column
  • Font Size - Change the font size of the table (8-24)

Right-Click Context Menu

  • Right-click on any cell to access quick row/column operations
  • All insert and delete operations are available from the context menu

Editing Cells

  • Click on a cell to select it
  • Double-click or press Enter to edit the cell content
  • Double-click column headers to rename them

Status Bar

  • Shows the number of rows and columns
  • Displays current encoding and separator
  • Shows selected cell coordinates in brackets [column,row]

Tips

  • Your language preference is automatically saved
  • You can resize columns by dragging the column borders
  • Files can be opened by dragging and dropping them into the window
  • The application supports up to 50 undo/redo levels
